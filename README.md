# PuyoPuyoConsole

Win32 콘솔 환경에서 직접 만든 렌더링 엔진 위에 구현한 C++ 뿌요뿌요 스타일 퍼즐 게임입니다.  
단순히 게임 규칙만 구현한 것이 아니라, 콘솔 더블 버퍼링, 입력 처리, 액터/레벨 구조까지 직접 설계해 작은 게임 프레임워크 형태로 제작했습니다.

## 프로젝트 소개

이 프로젝트는 `Engine`과 `Game`을 분리해 구성한 콘솔 게임 프로젝트입니다.

- `Engine`
  - Win32 콘솔 API 기반 렌더링
  - 더블 버퍼링
  - 입력 처리
  - 액터 / 레벨 생명주기 관리
- `Game`
  - 싱글 플레이
  - 2인 멀티 플레이
  - 메뉴, 일시정지, 게임 오버 상태 전환
  - 연쇄, 점수, 방해 뿌요 규칙 구현

포트폴리오 관점에서 이 프로젝트의 핵심은 다음 세 가지입니다.

- 콘솔 환경에서도 깜빡임을 줄이는 렌더링 구조를 직접 구현했습니다.
- 게임 로직과 엔진 레이어를 분리해 코드 구조를 설계했습니다.
- 싱글 플레이뿐 아니라 멀티 플레이와 공격/상쇄 시스템까지 확장했습니다.

## 개발 목적

이 프로젝트는 단순한 클론 제작보다 아래 역량을 보여주기 위해 만들었습니다.

- 게임 루프와 렌더링 파이프라인 이해
- 오브젝트 생명주기와 레벨 구조 설계
- 자료구조 기반 퍼즐 판정 로직 구현
- 확장 가능한 코드 구조로 기능 추가
- 사용자 입력, 상태 전환, 애니메이션 타이밍 처리

## 사용 기술

- C++
- Visual Studio 2022
- Win32 Console API
- Visual Studio Solution / MSVC v143 Toolset

## 주요 기능

### 1. 커스텀 콘솔 엔진

직접 구현한 `Engine` DLL이 게임의 기반입니다.

- 콘솔 화면 버퍼를 이용한 더블 버퍼링
- 프레임 기반 메인 루프
- 키 입력 상태 감지
- `Actor`, `Level` 기반 업데이트 / 렌더링 구조
- 설정 파일(`Settings/EngineSettings.txt`) 기반 화면 크기 / 프레임레이트 제어

### 2. 싱글 플레이

- 뿌요 2개가 한 쌍으로 생성
- 좌우 이동, 회전, 소프트 드롭
- 착지 후 그리드 반영
- 같은 색 4개 이상 연결 시 제거
- 중력 적용 후 연쇄 계산
- 점수에 따라 하강 속도 증가

### 3. 2인 멀티 플레이

- 플레이어별 독립 필드 운영
- 각자 다른 키 입력 처리
- 연쇄 점수 기반 공격 시스템
- 상대 필드에 방해 뿌요 생성
- 공격 상쇄 처리
- 패배 판정 및 멀티 게임 오버 화면 전환

## 구현 상세

### 콘솔 더블 버퍼링

일반 콘솔 출력은 프레임마다 화면이 심하게 깜빡이는 문제가 있습니다. 이를 해결하기 위해 `ScreenBuffer`를 사용해 두 개의 콘솔 버퍼를 번갈아 출력하도록 구현했습니다.

이 방식으로:

- 액터가 매 프레임 다시 그려져도 화면 흔들림이 줄어듭니다.
- 렌더링 책임을 `Engine` 내부로 모을 수 있습니다.
- 게임 로직은 화면 출력 방식보다 액터 상태 변화에 집중할 수 있습니다.

### 액터 / 레벨 구조

모든 게임 오브젝트는 `Actor`를 상속받고, 실제 씬 단위 관리는 `Level`이 담당합니다.

- `Actor`
  - 위치, 문자열 이미지, 색상, 생명주기 관리
- `Level`
  - 액터 추가 / 삭제 요청 처리
  - `BeginPlay`, `Tick`, `Render` 흐름 관리

이 구조 덕분에 메뉴 화면, 퍼즐 플레이 화면, 일시정지 화면, 게임 오버 화면을 각각 별도 레벨로 분리할 수 있었습니다.

### 뿌요 판정 로직

그리드는 배열로 관리하고, 같은 색 연결 판정은 BFS 방식으로 구현했습니다.

- 6 x 12 그리드 기준 탐색
- 상하좌우 인접 뿌요 검사
- 4개 이상 연결된 그룹 제거
- 제거 후 중력 재적용
- 연쇄 수, 색상 수, 그룹 크기에 따른 점수 보너스 계산

멀티 플레이에서는 여기에 방해 뿌요 처리까지 추가했습니다.

이 프로젝트에서 특히 보여주고 싶은 부분은 다음입니다.

- 싱글 플레이 구현 후 멀티 플레이로 확장한 점

## 프로젝트 구조

```text
PuyoPuyoConsole
├─ Engine/          # 콘솔 엔진 코드
├─ Game/            # 게임 로직 코드
├─ Includes/        # Engine 공개 헤더
├─ Lib/Engine/      # Engine 빌드 결과물
├─ Settings/        # 엔진 설정 파일
└─ PuyoPuyo_doubleBuffer.sln
```

## 실행 방법

### 요구 사항

- Windows
- Visual Studio 2022
- MSVC v143 Toolset

### 빌드

1. `PuyoPuyo_doubleBuffer.sln`을 엽니다.
2. `Game` 프로젝트를 시작 프로젝트로 설정합니다.
3. `Debug|x64` 또는 `Release|x64`로 빌드 후 실행합니다.

빌드 과정에서 `Engine` DLL과 라이브러리가 생성되고, `Game` 프로젝트가 이를 링크해 실행됩니다.

### 엔진 설정

`Settings/EngineSettings.txt`

```txt
width = 100
height = 35
framerate = 120
```

## 조작 방법

### 싱글 플레이

- 이동: `Left Arrow`, `Right Arrow`
- 소프트 드롭: `Down Arrow`
- 회전: `Q`, `E`
- 일시정지: `Esc`

### 멀티 플레이

#### Player 1

- 이동: `A`, `D`
- 소프트 드롭: `S`
- 회전: `Q`, `E`

#### Player 2

- 이동: `Left Arrow`, `Right Arrow`
- 소프트 드롭: `Down Arrow`
- 회전: `K`, `L`

## 주요 소스 위치

- 엔진 메인 루프: `Engine/Core/Engine.cpp`
- 콘솔 버퍼 렌더링: `Engine/Render/ScreenBuffer.cpp`
- 게임 상태 관리: `Game/Game/Game.cpp`
- 싱글 플레이 로직: `Game/Level/SinglePlayLevel.cpp`
- 멀티 플레이 로직: `Game/Level/MultiPlayLevel.cpp`
- 싱글 뿌요 액터: `Game/Actor/Puyo.cpp`
- 멀티 뿌요 액터: `Game/Actor/MultiPlayPuyo.cpp`
